# Vi@Po (Viable Pollen Analyzer)

Vi@Po is a smartphone-based digital phenotyping application designed for rapid and accurate assessment of pollen viability and germination.

---

## 🔬 Overview
Traditional pollen viability assessment relies on manual counting and data transcription, which are time-consuming and prone to human error.  
Vi@Po introduces a tap-based digital workflow that enables real-time annotation, automated counting, and direct data export.

---

## ✨ Features
- Tap-based marking of viable (green) and non-viable (red) pollen
- Real-time counting
- Automatic viability (%) calculation
- CSV export for analysis (Excel, R, Python)
- Compatible with multiple assays:
    - Acetocarmine staining
    - Fluorescein diacetate (FDA) test
    - TTC test
    - Alexander stain
    - In vitro pollen germination

---

## 📱 Platform
Android (Smartphone-based)

---

## 🚀 Installation
1. Clone this repository or download the APK
2. Install on an Android device
3. Open the app and start counting pollen

---

## 📊 Output
The app generates a CSV file containing:
- Sample ID
- Viable pollen count
- Non-viable pollen count
- Viability percentage
